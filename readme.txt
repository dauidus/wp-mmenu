Here are two examples of how you can use the shortcode:

[mmenu]example[/mmenu]
    (within your page content)

        **OR**

<a href="#mmenu">example</a>
    (in a template file)
    
These do the exact same thing.


...


You can add the css class `mm-fixed-top` or `mm-fixed-bottom` to any element and it should make it fixed.
